# 🤝 المساهمة في "تحدي الـ 45 يوم"

أولاً، شكراً لاهتمامك بالمساهمة! هذا المشروع يهدف لمساعدة الجميع على بناء عادات إيجابية، ومساهمتك تجعل التطبيق أفضل للجميع.

## 📋 كيف يمكنك المساهمة؟

يمكنك المساهمة بعدة طرق:
1. **الإبلاغ عن أخطاء (Bugs):** إذا وجدت مشكلة في التنبيهات أو نظام الألوان.
2. **اقتراح ميزات جديدة:** مثل إضافة رسوم بيانية للتقدم أو لغات إضافية.
3. **تحسين الكود:** إذا كان لديك تعديل يجعل الـ JavaScript أسرع أو الـ CSS أجمل.

## 🛠️ خطوات المساهمة (الجانب التقني)

1. قم بعمل **Fork** للمستودع.
2. أنشئ فرعاً جديداً لميزتك (Branch): `git checkout -b feature/NewHabit`.
3. قم بعمل **Commit** لتغييراتك: `git commit -m 'إضافة ميزة جديدة'`.
4. ارفع التغييرات لفرعك: `git push origin feature/NewHabit`.
5. افتح **Pull Request** واشرح ما الذي قمت بتعديله.

## 💡 قواعد كتابة الكود (Code Style)

بما أننا نستخدم تقنيات بسيطة لضمان عمل الموقع على التابلت والأجهزة الضعيفة:
- يفضل استخدام **Vanilla JavaScript** (بدون مكتبات خارجية).
- حافظ على استخدام **CSS Variables** لتسهيل دعم الـ Dark Mode.
- تأكد من أن الموقع يظل **Responsive** (متوافق مع الموبايل).

## 🛡️ ميثاق الأخلاق
نحن نقدر الاحترام المتبادل. يرجى أن تكون ردودك في الـ Issues والـ Pull Requests بناءة ومهذبة.
