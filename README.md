# habit-tracker
# 5 عادات لمدة 45 يوم 🌱 (45 Days Habit Tracker)

تطبيق ويب بسيط وعملي يساعدك على تتبع 5 عادات أساسية لمدة 45 يوماً بنظام تقييم بصري يعتمد على الألوان لتشجيع الاستمرارية والإنجاز.

## ✨ العادات المستهدفة في التحدي:
1. **المحافظة على الصلاة والأذكار.**
2. **المذاكرة (ساعتين على الأقل يومياً).**
3. **حفظ الورد اليومي من القرآن.**
4. **صلاة الفجر جماعة في المسجد.**
5. **تعلم التجويد.**

## 🚀 المميزات:
- **نظام تقييم ذكي:** يتغير لون اليوم بناءً على عدد العادات المنجزة (مثالي، جيد، ضعيف).
- **الوضع الليلي (Dark Mode):** مريح للعين للاستخدام قبل النوم.
- **تتبع التقدم:** شريط تقدم (Progress Bar) يوضح نسبة الإنجاز الإجمالية.
- **تنبيهات مخصصة:** إمكانية ضبط وقت للتذكير اليومي.
- **متوافق مع الشاشات:** يعمل بشكل ممتاز على التابلت والموبايل (Responsive Design).
- **صديق لمحركات البحث:** مجهز بوسوم SEO و Sitemap للأرشفة على Google.

## 🛠️ التقنيات المستخدمة:
- HTML5
- CSS3 (Custom Variables & Flexbox/Grid)
- JavaScript (LocalStorage for data persistence)
